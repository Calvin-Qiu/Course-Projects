This is an implementation of the Game of Amazons in Java, whose main features include the following:  
路 Includes a GUI  
路 Supports games between either two human players or a human player and an A.I  
路 The A.I. player uses alpha-beta pruning to conduct fast search for optimal moves in the game tree  
路 The A.I player achieved a 63% chance of wining over 200+ games with human players
